NewsAnalysisAPI

Proje haber içeriklerini veritabanından çekip analiz eder.

Kurulum:
- appsettings.json'da veritabanı bağlantısını yapın.
- dotnet run ile projeyi çalıştırın.

API uç noktaları:
- GET /api/news
- GET /api/news/{id}
- POST /api/analysis

Swagger arayüzü:
http://[sunucu-adresi]:[port]/swagger/index.html
